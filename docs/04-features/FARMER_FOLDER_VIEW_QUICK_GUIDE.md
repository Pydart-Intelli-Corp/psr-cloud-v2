# Farmer Folder View - Quick Guide

## Overview
The Farmer Management page now offers two ways to view your farmers:
- **Folder View**: Farmers organized by society (like folders on your computer)
- **List View**: All farmers in one grid (traditional view)

---

### To Use Folder View

## Working with Folders

### Opening a Society Folder
**Click on any society folder header** to see the farmers in that society.

**What you'll see**:
- Society name and ID
- Number of active farmers (green dot)
- Number of inactive farmers (red dot)
- Total farmer count

### Closing a Society Folder
**Click on an open folder header** to close it and hide the farmers.

### Expand All Folders at Once
Click the **"Expand All"** button to open every society folder.

### Collapse All Folders at Once
Click the **"Collapse All"** button to close all folders.

## Using Filters with Folder View

### All filters work the same way:
- **Status Filter**: Shows only farmers with selected status in each folder
- **Society Filter**: Shows only one society folder
- **Machine Filter**: Filters farmers within each folder
- **Search**: Highlights matching farmers inside folders

### Example:
1. Select "Active" status filter
2. Switch to Folder View
3. Each folder shows only active farmers
4. Inactive farmers are hidden

---

## Selecting Farmers in Folder View

### Individual Selection
- Check the box on any farmer card
- Works across multiple societies
- Selection persists when you expand/collapse folders

### Select All
- Clicks "Select All" checkbox
- Selects all visible farmers (respects current filters)
- Works across all societies

### Bulk Actions
- Delete selected farmers
- Change status for selected farmers
- Download selected farmers
All work the same in folder view!

## Common Tasks

### Task 1: Find All Farmers in a Specific Society
1. Switch to **Folder View**
2. Find the society name in the folder list
3. Click on that society folder to expand it
4. All farmers in that society appear

### Task 2: Check Active Farmers per Society
1. Switch to **Folder View**  
2. Look at the folder headers (no need to expand)
3. Green dot number shows active farmers
4. Do this for all societies quickly

### Task 3: Work with Multiple Societies
1. Click **"Expand All"** to open all folders
2. Scroll through societies
3. Work with farmers in any society
4. Click **"Collapse All"** when done

### Task 4: Focus on One Society
1. Use the **Society Filter** dropdown
2. Select the society you want
3. Only that society's folder appears
4. Expand it to see all farmers

### Task 5: Search Across All Societies
1. Enter your search query in the search box
2. Switch to **Folder View**
3. Expand folders to see matching farmers
4. Search highlighting shows matches

## Frequently Asked Questions

### Q: Can I still use search in folder view?
**A**: Yes! Search works exactly the same. Matching farmers are highlighted inside their society folders.

### Q: Do filters work in folder view?
**A**: Absolutely! All filters work the same way in both views.

### Q: Can I select farmers from different societies?
**A**: Yes! Expand multiple folders and select farmers from any society.

### Q: Will my view preference be remembered?
**A**: Currently, the page defaults to Folder View on load. Future updates may save your preference.

### Q: What happens to farmers with no society?
**A**: They appear in an "Unassigned" folder (ID: N/A).

### Q: Can I edit/delete farmers in folder view?
**A**: Yes! All actions (View, Edit, Delete) work the same in both views.

### Q: Does folder view affect downloads?
**A**: No. CSV/PDF downloads include the same data regardless of view mode.

### Q: How do I see all farmers at once?
**A**: Either switch to List View, or click "Expand All" in Folder View.
