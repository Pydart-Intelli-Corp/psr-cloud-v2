## üîß Combining Search with Filters

### Example 1: Active Machines by Operator
1. Set **Status** filter to "Active"
2. Search for **operator name** (e.g., "John")
3. See only active machines operated by John

### Example 2: Maintenance Machines in Society
1. Set **Society** filter to "Green Valley Dairy"
2. Set **Status** filter to "Maintenance"
3. Search for **machine type** (e.g., "AMCUPro")
4. See only AMCUPro machines under maintenance in that society

### Example 3: Find Machine by Location
1. Leave **Status** as "All"
2. Search for **location** (e.g., "Building A")
3. See all machines in Building A

## ‚ùì Common Questions

### Q: Why don't I see any results?
**A**: Check if:
- Your search term is spelled correctly
- Filters are too restrictive (try "All" status)
- The machine actually exists in your database

### Q: How do I search for multiple things?
**A**: Use filters for categories (status, society) and search for specific text

### Q: Can I search by date?
**A**: Not currently - use filters and manual sorting

### Q: Does search work with filters?
**A**: Yes! Search and filters work together

### Q: Is search case-sensitive?
**A**: No, search works regardless of uppercase/lowercase
