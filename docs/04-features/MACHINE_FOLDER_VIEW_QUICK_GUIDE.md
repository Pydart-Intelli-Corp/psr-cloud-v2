# Machine Folder View - Quick Guide

## Overview
The Machine Management page now offers two ways to view your machines:
- **Folder View**: Machines organized by society (like folders on your computer)
- **List View**: All machines in one grid (traditional view)

## Working with Folders

### Opening a Society Folder
**Click on any society folder header** to see the machines in that society.

**What you'll see**:
- Society name and ID
- Number of active machines (green dot)
- Number of inactive machines (red dot)
- Number of machines under maintenance (blue dot)
- Total machine count

### Closing a Society Folder
**Click on an open folder header** to close it and hide the machines.

### Expand All Folders at Once
Click the **"Expand All"** button to open every society folder.

### Collapse All Folders at Once
Click the **"Collapse All"** button to close all folders.

## Using Filters with Folder View

### All filters work the same way:
- **Status Filter**: Shows only machines with selected status in each folder
- **Society Filter**: Shows only one society folder
- **Machine Filter**: Filters specific machines within folders
- **Search**: Highlights matching machines inside folders

### Example:
1. Select "Active" status filter
2. Switch to Folder View
3. Each folder shows only active machines
4. Inactive and maintenance machines are hidden

## Common Tasks

### Task 1: Find All Machines in a Specific Society
1. Switch to **Folder View**
2. Find the society name in the folder list
3. Click on that society folder to expand it
4. All machines in that society appear

### Task 2: Check Machine Status per Society
1. Switch to **Folder View**  
2. Look at the folder headers (no need to expand)
3. Green/red/blue dot numbers show machine status counts
4. Do this for all societies quickly

### Task 3: Work with Multiple Societies
1. Click **"Expand All"** to open all folders
2. Scroll through societies
3. Work with machines in any society
4. Click **"Collapse All"** when done

### Task 4: Focus on One Society
1. Use the **Society Filter** dropdown
2. Select the society you want
3. Only that society's folder appears
4. Expand it to see all machines

### Task 5: Search Across All Societies
1. Enter your search query in the search box
2. Switch to **Folder View**
3. Expand folders to see matching machines
4. Search highlighting shows matches

### Task 6: Set Passwords for Multiple Machines
1. Expand the society folder
2. Click gear icon on first machine
3. Set passwords and save
4. Repeat for other machines
5. Password status updates immediately

## Frequently Asked Questions

### Q: Can I still use search in folder view?
**A**: Yes! Search works exactly the same. Matching machines are highlighted inside their society folders.

### Q: Do filters work in folder view?
**A**: Absolutely! All filters work the same way in both views.

### Q: Can I set passwords in folder view?
**A**: Yes! Click the gear icon on any machine card to manage passwords.

### Q: Will my view preference be remembered?
**A**: Currently, the page defaults to Folder View on load. Future updates may save your preference.

## Troubleshooting

### Issue: Folder won't expand
**Solution**: Click directly on the folder header area. Make sure you're not clicking on buttons within the header.

### Issue: Can't find a machine
**Solution**: 
1. Check your filters - they might be hiding the machine
2. Try searching for the machine ID
3. Switch to List View to see all machines

### Issue: Password settings not working
**Solution**: Make sure you're clicking the gear icon on the correct machine card. Password modal should appear.

### Issue: Counts don't match
**Solution**: Refresh the page to reload data. Check if filters are applied.
